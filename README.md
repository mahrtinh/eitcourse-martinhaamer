
# Public Transport & Daily Commute Analysis

This repository contains analysis of commute patterns using Quarto and R.

Mindmap of project idea: https://excalidraw.com/#json=LJu4JBwU35Qrxl-_oKJzp,zz1TjPKuGm65Dqppz8BASw

## Files
- `eitpaper.qmd` - Main analysis document
- `data/` - Spatial data files
- `images/` - Figures and visualizations

## How to run
1. Open `eitpaper.qmd` in RStudio
2. Install required packages: `sf`, `leaflet`, `dplyr`
3. Render the document
